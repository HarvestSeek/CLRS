    4.Red-Black trees

    1. 红黑树及其性质

    (1). 定义

     一棵红黑树是满足下列红黑性质 (red-black properties) 的二叉搜索树：
     1. 每个结点或为红色，或为黑色。
     2. 根结点是黑色的。
     3. 外部结点 NIL 是黑色的。
     4. 如果一个结点是红色的，那么它的两个子结点是黑色的。
     5. 对每一个结点，从该结点都所有后代叶结点的简单路径上，均包含相同数目的黑色结点。
     空结点视为外部结点（叶结点），并且为了方便处理边界条件，所有的空结点都由哨兵结点 NIL 代替。
     从某个结点 x 出发（不含该结点），到达一个外部结点的任一简单路径上的黑色结点数称为该结点的黑高度 (black-height)，记为 $bh(x)$。
     定义红黑树的黑高度为其根结点的黑高度。

    (2). 其他性质

    ① 在一棵以 $x$ 为根的(子)树中，设内部结点数为 $n$，则 $2^{bh(x)}-1\leq n \leq 2^{2×bh(x)}-1$。
     - 最少：全是黑结点；最多：任一路径上红黑交替。
    ② 从根结点到外部结点的任一条简单路径上，至少有一半的结点为黑色。
     - 由红黑性质 4 可得。
    ③ 从某结点 $x$ 到其后代外部结点的所有简单路径中，最长的一条的长度至多是最短的一条的 2 倍。
     - 由 ② 和红黑性质 5 可得。
    ④ 一棵有 $n$ 个内部结点的红黑树高度至多为 $2$·$lg(n+1)$。
     - 设树高为 $h$，由 ②，黑高度至少为 $h/2$，则 $n \geq 2^{h/2}-1$。两边取对数得 $h\leq 2$·$lg(n+1)$。

    2. 红黑树的操作

    (1). ROTATION

     红黑树的插入和删除操作可能会影响到树的红黑性质，此时通过修改结点颜色和修改树的指针结构维护这些性质，而修改指针结构通过旋转完成。

    void LeftRotate(RBTree* tree, Node *x) {
    Node *y = x->right;
    // step 1. turn y's left subtree
    x->right = y->left;
    if (y->left != tree->nil) {  
        y->left->parent = x;
    }
    // step 2. link x's parent to y
    y->parent = x->parent;
    if (x->parent == tree->nil) {
        tree->root = y;
    } else if (x == x->parent->left) {
        x->parent->left = y;
    } else {
        x->parent->right = y;
    }
    // step 3. link x and y
    x->parent = y;
    y->left = x;
}

    void RightRotate(RBTree* tree, Node *y) {
    Node *x = y->left;
    y->left = x->right;
    if (x->right != tree->nil) {
        x->right->parent = y;
    }
    x->parent = y->parent;
    if (y->parent == tree->nil) {
        tree->root = x;
    } else if (y == y->parent->left) {
        y->parent->left = x;
    } else {
        y->parent->right = x; 
    }
    y->parent = x;
    x->right = y;
}

    (2). INSERT

     调用 BST 的插入过程插入结点 z，为保持树结构的合理性，令 z 的左右孩子指向 nil，并置其为红色。这步操作可能导致红黑性质 2 和 4 其中的一个被破坏，下面分三种情况（在程序中为六种）讨论如何调整树结构。

    ① z 的叔结点 u 为红色
     此时 z、z.parent、u 为红色，z.parent.parent 为黑色。把 z.parent、u 着为黑色，把 z.parent.parent 着为红色，然后令 z = z.parent.parent，重复调整树结构的迭代过程。

    ② z 的叔结点 u 为黑色，z、z.parent、z.parent.parent 共线
     以 z.parent 是左孩子为例，此时 z、z.parent 为红色，u、z.parent.parent 为黑色。把 z.parent 着为黑色，z.parent.parent 着为红色，然后调用一次右旋 RIGHT-ROTATE(T, z.p.p)，保持了红黑树性质。

    ③ z 的叔结点 u 为黑色，且不共线
     以 z.parent 是左孩子为例，此时 z 是 z.parent 的右孩子。令 z 上移到 z.parent，然后对 _z 调用左旋操作，无需修改任意结点的颜色，然后按情况 ② 处理。


     最后，修改根结点的颜色，确保红黑性质 2 不被破坏。

    void Insert(RBTree *tree, Node *z) {
    Node *root = tree->root;
    Node *last = tree->nil;
    int key = z->key;
    while (root != tree->nil) {
        last = root;
        if (root->key > key) {
            root = root->left;
        } else {
            root = root->right;
        }
    }
    z->parent = last;
    if (last == tree->nil) {
        tree->root = z;
    } else if (last->key > key) {
        last->left = z;
    } else {
        last->right = z;
    }
    z->left = z->right = tree->nil;
    z->color = RED;
    InsertFixup(tree, z);
}

void InsertFixup(RBTree *tree, Node *z) {
    while (z->parent->color == RED) {
        if (z->parent == z->parent->parent->left) {
            Node *uncle = z->parent->parent->right; 
            if (uncle->color == RED) {
                z->parent->color = BLACK;
                uncle->color = BLACK;
                z->parent->parent->color = RED;
                z = z->parent->parent;
            } else if (z == z->parent->right) {
                z = z->parent;
                LeftRotate(tree, z);
            }
            z->parent->color = BLACK;
            z->parent->parent->color = RED;
            RightRotate(tree, z->parent->parent);
        } else {
            Node *uncle = z->parent->parent->left;
            if (uncle->color == RED) {
                z->parent->color = BLACK;
                uncle->color = BLACK;
                z->parent->parent->color = RED;
                z = z->parent->parent;
            } else if (z == z->parent->left) {
                z = z->parent;
                RightRotate(tree, z);
            }
            z->parent->color = BLACK;
            z->parent->parent->color = RED;
            LeftRotate(tree, z->parent->parent);
        }
    }
    tree->root->color = BLACK;
}

     因为树高 $O(lgn)$，所以过程 INSERT 插入元素的部分需要 $O(lgn)$ 时间；在过程 INSERT-FIXUP 中，当且仅当情况 ① 发生，while 循环才会重复进行，可能执行的总次数为 $O(lgn)$；因此，总共需要时间为 $O(lgn)$。
     此外，该程序所做的旋转次数不会超过 2。

    (3). DELETE

     类似于插入操作，先调用 BST 的删除过程，再分情况讨论如何调整树的结构。
     设要被删除的结点为 z，设 y 是因为删除或移动导致红黑性质被破坏的结点，设 x 是替换 y 的位置的新结点。对应于 BST 删除操作的三种情况，在前两种情况中，y 即为 z，x 是替换它的左右子树结点或空结点；在第三种情况中，因为替换 z 的结点会保持 z 的颜色，所以 y 是 z 右子树的后继，x 是 y 的右孩子或空结点。
     最后，依据 y 的颜色 y-original-color 判断是否调整树的结构。如果是红色，那么 y 不可能是根，且 x 一定是黑色，所以任何红黑性质都不会被破坏；如果是黑色，(可能)会发生

    • (i). 如果 y 是根，它的红孩子 x 成为新根，则破坏了红黑性质 2；
    • (ii). 如果在 y 被删除或移动后，x 和 x.parent 都是红色，则破坏了红黑性质 4；
    • (iii). 先前任何包含 y 的简单路径上的黑色结点数都少一。

    解决的办法是，视 x 的属性中有一重额外的黑色。如果 x 是一个红黑色结点，即其本身为红色，则直接把它置黑；如果 x 是一个双重黑色的结点，设 w 是它的兄弟结点，然后分四种情况（程序中为八种）讨论如何处理。（以 x = x.parent.left 为例）

    ① x 的兄弟结点 w 是红色的
     w 的两个孩子和 x.parent 是黑色的。先改变 w 和 x.parent 的颜色，然后对 x.parent 做左旋操作，w 的左孩子成为 new w。此时，x 具有双重黑色，new w 为黑色，进入情况 ②、③、④ 的讨论范围。
     对于子树 $\varepsilon$、$\xi$，交换颜色保证了红黑性质 5。


    ② x 的兄弟结点 w 为黑色，且 w 的两个子结点都是黑色的
     x、w 和 w 的两个孩子均为黑色的，x.parent 颜色未知。从 x 和 w 上去掉一重黑色，使得 x 只剩一重黑色、w 变为红色，并把这重黑色补偿到 x.parent 上，并令 x.parent = new x，重复迭代过程。
     如果情况 ② 是通过情况 ① 进入的，那么 new x 为红黑色，在代码的最后部分直接置黑，完成树结构的调整。

    ③ x 的兄弟结点 w 是黑色的，且 w 的左孩子为红色，右孩子为黑色
     x.p 颜色未知。改变（交换）w 和 w.left 的颜色，然后对 w 做右旋操作。此时，new w 为黑色，且 w 的右孩子是红色的，进入情况 ④ 处理。

    ④ x 的兄弟结点 w 为黑色，且 w 的右孩子为红色
     x.p 颜色未知。因为 w 会成为子树的新根，令 w.color = x.p.color，然后改变 w.right 的颜色，并置 x.p 为黑色，再对 x.p 做左旋操作，去掉了 x 上额外的一重黑色，并保持红黑性质不变。最后退出循环。


     在红黑性质被破坏的几种情形中，当 (ii). 发生，在过程 DELETE-FIXUP 中循环外被调整；删除的四种情况中的变换主要是为了调整 (iii). 造成的红黑性质 5 被破坏，每种情况均保证了变换前后根结点到子树 $\alpha$ ~ $\xi$ 的黑结点数不变。

    void Delete(RBTree *tree, Node *z) {
    Node *y = z;
    Node *x;
    Color yOrgColor = y->color;
    if (z->left == tree->nil) {
        x = z->right; 
        Transplant(tree, z, z->right);      
    } else if (z->right == tree->nil) {
        x = z->left;
        Transplant(tree, z, z->left);
    } else {
        y = Minimum(tree, z->right);
        x = y->right;
        if (y->parent == z) {
            x->parent = y;  
            // if z->right links to y directly and y->right links 
            // to nil, let nil->parent links to y
        } else {
            Transplant(tree, y, y->right);
            y->right = z->right;
            z->right->parent = y;
        }
        Transplant(tree, z, y);
        y->left = z->left;
        z->left->parent = y;
        y->color = z->color;  // shift color
    }
    if (yOrgColor == BLACK) {
        DeleteFixup(tree, x);
    }
}

    void DeleteFixup(RBTree *tree, Node *x) {
    while (x != tree->root && x->color == BLACK) {
        if (x == x->parent->left) {
            Node *w = x->parent->right;
            if (w->color == RED) {
                w->color = BLACK;
                x->parent->color = RED;
                LeftRotate(tree, x->parent);
                w = x->parent->right;
            }
            if (w->left->color == BLACK && w->right->color == BLACK) {
                w->color = RED;
                x = x->parent;
            } else {
                if (w->right->color == BLACK) {
                    w->left->color = BLACK;
                    w->color = RED;
                    RightRotate(tree, w);
                    w = x->parent->right;
                }
                w->color = x->parent->color;
                x->parent->color = BLACK;
                w->right->color = BLACK;
                LeftRotate(tree, x->parent);
                break;
            }
        } else {
            Node *w = x->parent->left;
            if (w->color == RED) {
                w->color = BLACK;
                x->parent->color = RED;
                RightRotate(tree, x->parent);
                w = x->parent->left;
            }
            if (w->left->color == BLACK && w->right->color == BLACK) {
                w->color = RED;
                x = x->parent;
            } else {
                if (w->left->color == BLACK) {
                    w->right->color = BLACK;
                    w->color = RED;
                    LeftRotate(tree, w);
                    w = x->parent->left;
                }
                w->color = x->parent->color;
                x->parent->color = BLACK;
                w->left->color = BLACK;
                RightRotate(tree, x->parent);
                break;
            }
        }
    }
    x->color = BLACK;
}

    void Transplant(RBTree *tree, Node *des, Node *src) {
    if (des == tree->root) {
        tree->root = src;
    } else if (des == des->parent->left) {
        des->parent->left = src;
    } else {
        des->parent->right = src;
    }
    // src may be T.nil
    src->parent = des->parent;
}
