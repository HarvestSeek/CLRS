    7.Segment trees

    1. 线段树的性质

    一棵线段树存储一个区间范围，每个结点中存储附加属性。

    • 对于一个区间为 $[l,\ r]$ 的结点，其左右子树对应的区间分别为 $[l,\ mid]$ 和 $[mid+1,\ r]$;
    • 由于区间向下二分，兄弟结点间的区间包含元素的差值至多为 $1$，因此线段树或为满二叉树 (总元素数为 $2$ 的幂)，或为最后一层未满的满二叉树，树高为 $O(\lg n)$；
    • 叶结点代表的区间是一个单点；
    • 对于线段树区间内的每个子区间 $i$，存在唯一确定的 $\log n$ 个结点，这些结点区间交集为空、并集为 $i$ 且 $n$ 值最小。

    2. 基本操作: 单点修改 + 区间查询

    (1). 建立一棵线段树

    线段树一旦建立，其结构不改变。
    当忽略最后一层的结点，线段树一定是一棵满二叉树，因此用数组表示。每个结点的属性通常仅与左右子树有关，这里以区间和为例。

    void build_tree(int index, int l, int r) {
    if (l == r) {
        tree[index] = sum[l];
        return;
    }
    int mid = (l + r) / 2;
    build_tree(index * 2 + 1, l, mid);
    build_tree(index * 2 + 2, mid + 1, r);
    tree[index] = tree[index * 2 + 1] + tree[index * 2 + 2];
}

    (2). 单点修改

    如果要修改某个单点的属性值，需要修改其到根的路径上每个结点的值。即终止条件为叶结点和不相交区间。

    void modify(int index, int l, int r, int point, int val) {
    if (l > point || r < point) 
        return;
    if (l == r && l == point) {
        tree[index] += val;
        return;
    }
    int mid = (l + r) / 2;
    modify(index * 2 + 1, l, mid, point, val);
    modify(index * 2 + 2, mid + 1, r, point, val);
    tree[index] += val;
}

    (3). 区间查询

    给定线段树区间的一个子区间，查询该区间内的属性信息。查询是一个遍历二叉树的过程，根据线段树的性质，可将查询分为三种情况：

    • 查询区间与当前区间无交集，则不再向下遍历并返回 0；
    • 查询区间包含了当前区间，则不再向下遍历并返回当前区间的属性；
    • 查询区间与当前区间有交集，则向左右依次递归继续查询，结束后返回二个查询结果的和。

    即终止条件为不相交区间和子集区间。

    int query(int index, int lc, int rc, int ls, int rs) {
    if (rs < lc || ls > rc) 
        return 0;
    if (lc >= ls && rc <= rs)
        return tree[index];
    int mid = (lc + rc) / 2;
    return query(index * 2 + 1, lc, mid, ls, rs) + query(index * 2 + 2, mid + 1, rc, ls, rs);
}

    3. 其他操作

    (1). 区间修改 + 单点查询

    如果要同时在某个区间内所有单点的属性上同时加上某值，那么就需要更新 $n$ 棵子树内的所有结点的属性。为提升效率，沿用区间查询的过程，定义一个额外的属性，且只更新查询到的 $\log n$ 个结点的这个属性即可：

    void modify(int index, int lc, int rc, int ls, int rs, int val) {
    if (rs < lc || rc < ls) 
        return;
    if (lc >= ls && rc <= rs) {
        addsum[index] += val;
        return;
    }
    int mid = (lc + rc) / 2;
    modify(index * 2 + 1, lc, mid, ls, rs, val);
    modify(index * 2 + 2, mid + 1, rc, ls, rs, val);
}
    如果单点查询的对象在发生修改的区间内，则回溯时它一定会路过此区间，只需要在返回时加上这个额外属性。

    int query(int index, int l, int r, int point) {
    if (l == r)
        return tree[index] + addsum[index];
    int mid = (l + r) / 2;
    if (point <= mid)
        return query(index * 2 + 1, l, mid, point) + addsum[index];
    else
        return query(index * 2 + 2, mid + 1, r, point) + addsum[index];
}

    void modify(int index, int lc, int rc, int ls, int rs, int val) {
    if (ls <= lc && rs >= rc) {
        addsum[index] += val;
        return;
    }    
    tree[index] += ((min(rs, rc) - max(lc, ls) + 1) * val);   
    int mid = (lc + rc) / 2;
    if (ls <= mid)
        modify(index * 2 + 1, lc, mid, ls, rs, val);
    if (rs > mid) 
        modify(index * 2 + 2, mid + 1, rc, ls, rs, val);
}

int query(int index, int lc, int rc, int ls, int rs) {
    if (ls <= lc && rs >= rc) 
        return tree[index] + addsum[index] * (rc - lc + 1);
    int mid = (lc + rc) / 2;
    int res = (min(rc, rs) - max(lc, ls) + 1) * addsum[index];
    if (ls <= mid)
        res += query(index * 2 + 1, lc, mid, ls, rs);
    if (rs > mid)
        res += query(index * 2 + 2, mid + 1, rc, ls, rs);
    return res;
}
